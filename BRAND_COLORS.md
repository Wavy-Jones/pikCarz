## Color Philosophy

**Bright Red (#FF4545):**
- Energetic and attention-grabbing
- Represents speed, passion, automotive excitement
- Used sparingly for maximum impact
- Perfect for CTAs, prices, badges

**White (#FFFFFF):**
- Clean, elegant, professional
- High contrast on dark backgrounds
- Modern and sophisticated

**Dark Charcoal (#1F2128):**
- Premium, upscale feeling
- Makes colors pop
- Reduces eye strain
- Professional automotive industry standard

## CSS Variables

```css
:root {
  --primary:     #1F2128;  /* Dark background */
  --surface:     #2A2D35;  /* Cards, surfaces */
  --card:        #353841;  /* Elevated cards */
  --accent:      #FF4545;  /* Bright red - primary brand color */
  --accent-glow: rgba(255, 69, 69, 0.15);
  --accent-dark: #E63535;  /* Darker red for hovers */
  --white:       #FFFFFF;
  --muted:       #9CA3AF;  /* Gray text */
  --border:      rgba(255,255,255,0.08);
}
```

## Usage Examples

### Buttons
- **Primary CTA:** White text on bright red background
- **Secondary:** White border on transparent, hover fills with red glow
- **Ghost:** Gray text, subtle hover

### Cards
- Dark charcoal background (#353841)
- White text for titles
- Red accents for make/price
- Subtle border glow on hover (red tint)

### Typography
- **Headings:** White
- **Body:** Light gray (#9CA3AF)
- **Highlights:** Bright red
- **Links:** Bright red

## Accessibility

✅ **WCAG AA Compliant:**
- White on Dark Charcoal: 15.8:1 contrast
- Bright Red on Dark Charcoal: 4.9:1 contrast
- Red buttons use white text: 5.7:1 contrast
