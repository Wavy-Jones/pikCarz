# pikCarz 🚗
### South Africa's Vehicle Marketplace — Powered by Cubeas

> A subscription-based vehicle marketplace platform where individuals and dealers can list and sell new cars, used cars, motorbikes, trucks and more.

---

## 🌐 Live Preview
> [https://wavy-jones.github.io/pikcarz](https://wavy-jones.github.io/pikcarz)

## 📁 Project Structure

```
pikcarz/
├── index.html        # Homepage with hero, featured listings & search
├── browse.html       # Browse page with filters & vehicle grid
├── about.html        # About pikCarz & Cubeas
├── contact.html      # Contact form & dealer enquiry
├── css/
│   └── styles.css    # Complete design system & responsive styles
└── js/
    └── main.js       # Mobile menu, animations, form handling
```

---

## ✨ Features

- **Dark premium UI** — navy & orange design system
- **Vehicle categories** — New Cars, Used Cars, Motorbikes, Trucks, Other
- **Advanced filter sidebar** — make, model, year, price, province, fuel, transmission
- **Dual user types** — Private sellers & Verified Dealers
- **Responsive design** — mobile-first, works on all screen sizes
- **Animated hero** — SVG car illustration with floating animation
- **Scroll animations** — fade-in on scroll, counter animations
- **Mobile menu** — full-screen overlay navigation
- **Save/wishlist** — heart button on each listing card
- **Contact form** — with submit animation & success state

## 🚀 Getting Started

No build tools or dependencies required. Just open in a browser:

```bash
# Clone the repo
git clone https://github.com/wavy-jones/pikcarz.git

# Open in browser
open pikcarz/index.html
```

Or simply double-click `index.html`.
